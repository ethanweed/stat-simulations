Shapiro-Wilk Test: False Negatives Simulation

Ethan Weed

3/24/2020

The Shapiro-Wilk test is used to evaluate whether a sample is likely to come from a normal distribution. The null hypothesis is that the sample is drawn from a normal distribution, thus a p < alpha indicates that the sample was drawn from a non-normal distribution. This simulation estimates distributions of percentages of p-values > 0.05 for the Shapiro-Wilk test when samples are drawn from a uniform distribution. This represents the frequency with which we could expect that the Shapiro-Wilk test would lead to the conclusion that the data are normally distributed, when in fact they are uniformally distributed. Data shown for sample sizes of 10, 30, 50, and 75 participants. Percentage estimates are bootstrapped over 1000 iterations. The final plot shows an example of what the data being tested looks like. Labels in plots show mean estimated percentage of false negatives.

library(ggplot2)
library(cowplot)

set.seed(42)


FalseNegative <- function(N){
  s <- NULL
  for (i in 1:1000){
    m <- NULL
    for (j in 1:100){
      data <- runif(N)
      a <- shapiro.test(data) 
      p <- a$p.value
      if (p > 0.05){
        m <- append(m,p)
      }
    }
    s <- append(s,length(m))
  }
  
  x <- seq(1:length(s))
  
  df <- data.frame(x,s)
  mu <- mean(df$s)
  mu <- round(mu, digits = 1)
  
  ggplot(df, aes(s)) +
    geom_density() +
    theme_cowplot() +
    #geom_vline(xintercept = mu) +
    geom_label(aes(x = 25, label = paste0(mu, " %"), y = 0.18)) +
    xlim(0,100) +
    ylim(0,.2) +
    labs(x = "percent",
         title = paste0(N, " Participants"))
}


data_plot <- function(N){
  points <- runif(N)
  participant <- seq(1:length(points))
  df_points <- data.frame(participant, points)
  ggplot(df_points, aes(participant, points)) +
    geom_point() +
    theme_cowplot() +
    labs(title = paste0(N, " Participants"))
  }


p1 <- FalseNegative(10)
p2 <- FalseNegative(30)
p3 <- FalseNegative(50)
p4 <- FalseNegative(75)
p5 <- FalseNegative(100)

p6 <- data_plot(50)

p_grid <- plot_grid(p1, p2, p3, p4, p5, p6)
